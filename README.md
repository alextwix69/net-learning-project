# net-learning-project

Учебный C++ проект для пошагового изучения сетей и протоколов.

## Структура

- `include/net/` - публичные заголовочные файлы.
- `src/common/` - общий код для сетевых примеров.
- `src/tcp_echo_server/` - будущий TCP echo server.
- `src/tcp_echo_client/` - будущий TCP echo client.
- `src/udp_ping/` - будущий UDP пример.
- `src/http_client/` - будущий HTTP client.
- `tests/` - будущие тесты.
- `docs/` - заметки и план обучения.
